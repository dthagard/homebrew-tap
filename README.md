# Homebrew Tap

Homebrew formulae for [dthagard](https://github.com/dthagard) projects.

## Installation

```
brew tap dthagard/tap
```

## Available Packages

| Package | Description |
|---------|-------------|
| [tforganize](https://github.com/dthagard/tforganize) | A CLI tool to sort and organize Terraform files |

### tforganize

```
brew install dthagard/tap/tforganize
```
